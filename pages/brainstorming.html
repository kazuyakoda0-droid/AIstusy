アイデア出しの壁打ち
"考える相手" が、いつでもいる。

AIに答えだけを求めない。「私の考えを批判して」「私に質問して」と頼めば、思考のパートナーになる。一人で煮詰まる時間を、対話の時間に変える。

Concept
In one sentence

"答えをくれるAI" ではなく、"質問してくるAI"。
使い方次第で、頭の中の整理係になる。

Diagram 01

2つのモード — 順引き vs 逆引き

「アイデアちょうだい」（順引き）と「私に質問して」（逆引き）。後者を使いこなすかで、生産性が変わる。

Mode 01 ／ 順引き

"案をくれ"

「新サービスのネーミング案を10個出して」「新人向けの研修テーマのアイデアを」。AIが候補を列挙してくれる。広く浅く出すのが得意。

→ 数を出すフェーズで有効

Mode 02 ／ 逆引き

"私に質問して"

「ターゲット設定で悩んでいる。私に3つ質問をして、絞り込ませて」。AIがファシリテーターとなり、人間の思考を引き出してくれる。

→ 自分の考えを言語化するフェーズで有効

Diagram 02

逆引きプロンプトの実例

企画のターゲット設定で詰まっているとき、AIに「質問させる」と、こうなる。一人で堂々巡りしていた思考が、対話の中で整理されていく。

User
新企画のターゲット設定で悩んでいます。私に3つ質問をして、ターゲットを絞り込ませてください。
AI
承知しました。順番にお伺いします。

Q1: この企画で、ユーザーが「これまで諦めていた・我慢していた」ことは何ですか？
User
朝の身支度の時間がなくて、スキンケアを諦めている人が多いと感じています。
AI
なるほど。
Q2: その人たちは、なぜ "時短化されたスキンケア商品" ではなく、御社の企画を選ぶべきだと考えていますか？
User
…そう言われると、差別化が弱い気がしてきました。
AI
良い気づきです。
Q3: 時間がない朝に "1ステップで終わる" 以外で、ユーザーの本音は何でしょう？（例：罪悪感を消したい、自己肯定感を上げたい、誰かに褒められたい）
User
"忙しくても自分を大切にできている" という感覚かもしれません。
AI
3つの回答をまとめると、ターゲットは「時短を求める層」ではなく「忙しさの中で自己肯定感を保ちたい20代後半〜30代女性」ではないでしょうか。差別化は時間ではなく "セルフケアの感情価値" で考えると、企画が立ちやすくなりそうです。
AIは 答えを持っていない。
しかし、人間の中にある答えを引き出す手助けはできる。
Templates

壁打ちの定番プロンプト

「ただアイデアちょうだい」を卒業すると、AIは一気に頼れる相棒になる。

01

批判してもらう

「以下の企画書を、厳しめのマーケティング部長の視点で批判してください。具体的に5つ。」
→ 自分では見落とす欠点を、容赦なく指摘してくれる。

02

異なる視点から見てもらう

「この提案を、(a)新人エンジニア、(b)経理部長、(c)エンドユーザー の3つの視点で評価してください。」
→ 一つの企画を多角的に検証できる。

03

反対意見を出させる

「私の意見：○○ですが、これに最も鋭く反論する人なら、何と言いますか？」
→ 反対意見を先に潰せる。後の説得材料になる。

04

質問でファシリテートさせる

「私に5つ質問してください。一つずつ答えるので、最後にまとめてください。テーマは○○です。」
→ AIが取材者・ファシリテーターとして、思考を整理してくれる。

Plain Language

用語解説 — むずかしい言葉をやさしく

このページに出てくる専門用語を、業務や日常に置き換えて理解する。

壁打ち Sparring / Bouncing Ideas

相手に話しながら、自分の考えを整理していくこと。ホワイトボードの前で同僚に話しながら考えがまとまる、あの感覚。AIが相手になる。

発散と収束 Divergence / Convergence

アイデア出しの2段階。たくさん出す→絞り込む、を繰り返す。"とにかく100個出す" → "上位3つを深掘り" のリズム。

認知バイアス Cognitive Bias

人間が無意識に陥る思考の偏り。AIに代わりに指摘させると気付ける。"前例があるからこれでいい" "上司が好きそうな案" など、無自覚な思い込み。

逆引きプロンプト Reverse Prompting

"あなたから私に質問して" とAIに会話を主導させる手法。考えがぼんやりしているとき、AIに質問させて自分の頭の中を整理する。
