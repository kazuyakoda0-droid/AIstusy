データ処理・Excel連携
関数を、言葉で書く。

「VLOOKUP の入れ子で、月別・部門別の売上を集計したい」。これまでは検索して試行錯誤していた作業を、自然言語で依頼するだけで、関数もマクロも一発で出てくる。

Concept
In one sentence

"こういう集計がしたい" と言葉で頼めば、
関数とマクロが返ってくる。Excelの民主化。

Diagram 01

自然言語 → Excel関数

「こうしたい」を日本語で書くだけで、対応する関数を返してくれる。複雑な入れ子や、IFERROR等のエラーハンドリングも一緒に組み込んでくれる。

Request 01 ／ 自然言語
A列の商品コードをもとに、別シートの単価表（B列＝コード、C列＝単価）から単価を持ってきたい。コードがなかったら "未登録" と表示して。
→
Output ／ Excel関数
=IFERROR(VLOOKUP(A2, 単価表!B:C, 2, FALSE), "未登録")
Request 02 ／ 自然言語
売上データから「2024年の月別売上合計」を出したい。日付はB列、売上はC列にある。
→
Output ／ Excel関数
=SUMPRODUCT((YEAR($B$2:$B$1000)=2024)*(MONTH($B$2:$B$1000)=ROW()-1)*$C$2:$C$1000)
Request 03 ／ 自然言語
A列の文字列から、最初の括弧（）の中身だけを取り出したい。
→
Output ／ Excel関数
=MID(A2, FIND("（",A2)+1, FIND("）",A2)-FIND("（",A2)-1)
Diagram 02

マクロ（VBA）も書ける

複数シートをまたぐ集計や、定型作業の自動化。プログラミング未経験でも、やりたいことを日本語で書けばVBAコードが返ってくる。

Request ／ 自然言語
フォルダ内の全Excelファイルを開き、それぞれの「合計」シートのA1セルの値を、新規ブックにファイル名と一緒に転記して保存するマクロを作って。
Sub CollectTotals()
  Dim folder As String, file As String
  Dim wb As Workbook, summary As Workbook
  Set summary = Workbooks.Add
  folder = "C:\reports\"
  file = Dir(folder & "*.xlsx")
  Dim i As Long: i = 1
  Do While file <> ""
    ' 各ファイルを開いて合計シートのA1を転記
    Set wb = Workbooks.Open(folder & file)
    summary.Sheets(1).Cells(i, 1).Value = file
    summary.Sheets(1).Cells(i, 2).Value = wb.Sheets("合計").Range("A1").Value
    wb.Close False
    i = i + 1: file = Dir
  Loop
  summary.SaveAs folder & "summary.xlsx"
End Sub

Workflow

失敗しないコツ — データ仕様を先に伝える

関数やマクロが「動かない」原因のほとんどは、データの形（列の位置、データ型、シート名）を伝え忘れること。

01

シート構成・列構成を必ず伝える

「シート名は『売上』、A列＝日付、B列＝顧客名、C列＝金額、1行目はヘッダー」。これを書くだけで的中率が大きく上がる。サンプルデータを5行貼るのも有効。

02

「動かない」をそのまま伝えて修正依頼

エラーメッセージや実際の出力をコピペで送ると、AIが自己修正案を返してくる。「#REF!エラーになる」「思った値と違って○○になる」と具体的に。

03

機密データはマスキングする

顧客名・金額の生データをそのまま貼らない。「顧客A、顧客B」「100、200」など、構造は保ったままダミーデータにして渡す。セキュリティの基本。

Plain Language

用語解説 — むずかしい言葉をやさしく

このページに出てくる専門用語を、業務や日常に置き換えて理解する。

関数 Function

Excelの「=SUM(A1:A10)」のような、決まった計算式。AIに「こういう集計をしたい」と日本語で伝えれば、関数を書いてくれる。

マクロ／VBA Macro / VBA

Excelで繰り返し作業を自動化する小さなプログラム。これまでは情シスや詳しい人しか書けなかったが、AIが代わりに書ける。

Python Python

今もっとも普及している、データ処理が得意なプログラミング言語。ExcelでもAIで扱える時代になり、ますます身近になっている。

自然言語 Natural Language

人間の普通の話し言葉・書き言葉。"このシートで都道府県ごとの売上を出して" のような、普通の日本語のこと。
